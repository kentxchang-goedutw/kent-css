# 課表排課系統 使用手冊 (User Manual)

本系統是一套專為學校教務處設計的**高效網頁版排課系統**，採用純瀏覽器端前端架構運行，無須安裝任何後端資料庫，即開即用。系統內建自動排課衝突規避演算法（CSP）、智慧同節衝堂檢查，並支援將課表匯出為高解析度 A4 PDF 與標準 Word (DOCX) 格式。

> [!NOTE]
> **AI 開發宣告**：本排課系統之全套介面、功能邏輯、自動排課演算法與匯出格式排版，皆是使用 Google 的 **Gemini 3.5 Flash** 大型語言模型輔助編譯與設計製作而成。

---

## 🚀 系統核心特色

1. **單一網頁架構 (Single-File App)**：所有邏輯、樣式與資料處理皆整合在 `index.html` 內，下載至本機即可離線使用。
2. **防呆限制性規避演算法 (CSP)**：在毫秒間自動計算並填入最佳課表，自動避開班級不上課、教師不可排課及教師同節衝堂。
3. **上方橫式分頁標籤**：引導式排課流程，1 到 4 步一目了然。
4. **雙向對照協同排課**：針對因衝堂未排入的課程，提供直觀的雙向對照面板，並標示出最適合排入的「✨ 完美時段」。
5. **支援多重匯出**：一鍵將全校班級、教師個人課表批次匯出為 Landscape A4 PDF 或真實 Microsoft Word (.docx) 檔案。

## 📖 引導式四步驟使用指南

開啟網頁後，請遵循頂部橫式選單的四個步驟依序操作：

### 步驟 1：年級與班級設定
* **新增年級與班級**：點擊「新增年級」，輸入年級名稱（例如：七年級）。接著在下方輸入班級名稱（例如：1班、2班），點擊「新增班級」將其加入列表中。
* **不上課時段設定**：每個年級右側都有「設定不上課時段」按鈕。點擊後會開啟時間框架格，將特定時間點擊設為紅色 `❌ 不上課`（例如：星期五下午最後一節），排課時系統便會自動避開此時段。

### 步驟 2：時間框架設定
* **每週天數與節數**：設定每週上課天數（預設 5 天）及每天的授課節數（預設 7 節）。
* **教師不可排課時段**：在此處您可以設定教師的公差、兼職、或者個人不便排課的時間。點擊教師右側的「設定不可排課時段」，在開啟的視窗中點擊特定格子設為 `❌ 不可排` 即可。

### 步驟 3：課程計畫與配課
本步驟分為三個子分頁，供您進行排課核心資料的建置：
1. **1. 課程計畫**：
   * 設定各年級各學科的每週授課節數（例如：七年級國語文每週 5 節、數學 4 節）。
   * 支援**「拆分」功能**：例如將「彈性課程 2 節」拆分為「彈性課程A 1節」與「彈性課程B 1節」，以利指派給不同的任課老師。
2. **2. 任課教師 & 導師**：
   * 建立全校教師名冊，並可在此指派各班級的「導師」職務。
3. **3. 手動拖曳配課**：
   * 系統將自動列出左側「待分配課程卡片」。
   * 您可以直接將課程卡片拖曳到右側的教師卡片中完成指派；亦可勾選多門課程，在右側下拉選單中選擇教師執行**「批次指派」**。
   * 如有指派錯誤，滑鼠移至教師卡片內的學科上，點擊紅色的**「退回」**，該課程即會歸還至待分配列表中。

### 步驟 4：執行自動排課
* 點擊 **`🚀 開始自動排課`**，系統將在 1.5 秒內啟動回溯 CSP 衝突規避演算法進行全校課表計算。
* 計算完畢後，系統會自動列出「已排入」與「未排入（有衝突）」的課程數據統計。

## 🛠️ 排課後手動調整與協同排課

自動排課完成後，您可以透過兩種直觀的視覺化方式微調課表：

### A. 課表儲存格直接點擊調整
在班級或教師課表視圖中，直接點擊任何一個時間格子：
* **空堂與清空**：點擊「🗑️ 清空此節課程設定（設為空堂）」，能立即清空該節。
* **指派新課程**：下方會顯示「指派該班課程給特定教師」列表。**系統已自動過濾，只會顯示「尚未排滿」的學科**，避免超額排課。
* **衝堂防呆機制**：若您選取的課程其任課教師在該時段已有其他授課，系統會發出**「指派衝突！王老師此時段已在其他班級授課」**的警告並阻擋指派。

### B. 雙向對照協同排課 (解決衝突剩餘課程)
當自動排課結束後，若有少數因硬性限制（如教師不可排課與衝堂重疊）未能排入的課堂，會在課表上方顯示**「因衝突未能自動排入的課程卡片」**：
1. 點擊卡片上的 **`🎯 點此開啟雙向對照排入`**，會開啟大型協同排課視窗。
2. 左側會呈現**「該班級的週課表」**，右側呈現**「該任課教師的週課表」**。
3. 系統會自動比對雙方時間，用醒目的綠色標示出兩邊皆是空堂的 **`✨ 完美時段`**。
4. 點選完美時段，並在下方點擊**「🎯 填入此未排課程」**即可一鍵完美填入！
5. **每週節數上限防護**：若課程已排滿每週上限（例如：數學每週設定 2 堂，目前已排 2 堂），系統會阻擋超額排入，保護每週授課上限不超支。

## 💾 資料備份與還原

為了避免瀏覽器關閉或重新整理導致輸入的資料遺失，請妥善利用系統頂部的備份功能：
* **💾 備份目前計畫**：將所有輸入的年級、時間、課程計畫、配課關係與目前的課表狀態打包，匯出成一個 `排課計畫_日期.json` 檔案存於本機。
* **📂 載入計畫備份**：重新開啟網頁時，點擊此按鈕載入先前備份的 `.json` 檔案，便能立即恢復工作進度。

## 🖨️ 課表下載與列印規格說明

系統提供極具現代感的黑白高對比排版，課表表格高度經精密預算調整，保證單頁呈現不溢出：

1. **📥 下載課表 PDF**：
   * 點擊綠色的「📥 下載課表 PDF」按鈕。
   * 將批次生成 Landscape (橫式) A4 PDF，班級課表以淡灰色呈現已排課程，無文字干擾不可排時段。
   * **列印排版優化**：包含行分割防截斷 (Avoid Row Break) 及 135mm 的安全頁高預算，確保一班一頁，列印不跨頁。
2. **📝 下載課表 Word (DOCX)**：
   * 點擊按鈕，系統將透過 UMD 套件在瀏覽器端直接封裝生成真實的 Microsoft Word `.docx` 檔案下載。
   * 格式相容於 Microsoft Word、WPS 及 LibreOffice 等主流文書軟體，開啟不會是空白或 HTML 拼貼檔。
